# Guia de Uso de Assets — Grupo Comsteel

> Como escolher, usar e combinar os assets deste Design System em materiais de comunicação.

---

## 01_Logos

### Quando usar cada versão

| Situação | Logo Construção | Logo Tiny House |
|---|---|---|
| Material sobre obras/casas/MCMV | ✅ | ❌ |
| Material sobre Tiny Houses/módulos | ❌ | ✅ |
| Material institucional do Grupo | ✅ (prioritário) | Secundário |
| Licenciamento | ✅ (com menção ao grupo) | Conforme produto |
| Co-branding com parceiro | Sempre manter zona de exclusão | Sempre manter zona de exclusão |

### Regras de aplicação
- Fundo escuro (`#000000` ou `#3a240d`) → usar versão clara do logo
- Fundo claro (branco, off-white, bege) → usar versão escura do logo
- Tamanho mínimo digital: 120px de largura
- Tamanho mínimo impresso: 25mm de largura
- ❌ Nunca colocar os dois logos juntos sem aprovação da diretoria

## 02_Renders_Tiny_House

### Seleção de imagens por objetivo

| Objetivo do material | Renders recomendados | Observação |
|---|---|---|
| Instagram/Redes sociais | Externos com natureza ao fundo | Alta saturação, luz dourada |
| Catálogo/Proposta comercial | Externos + internos do modelo específico | Mostrar pelo menos 1 externo e 1 interno |
| Apresentação para investidor | Externos + dado de rendimento sobreposto | Evitar internos muito pequenos |
| Landing page | Render frontal como hero + galeria com internos | |
| WhatsApp/DM inicial | 1 render externo impactante + 1 interno | Máximo 2 imagens por mensagem |
| Anúncio pago (Meta/Google) | Render externo com contexto de natureza | Evitar fundo neutro/vazio |

### Modelos por perfil de cliente

| Perfil do lead | Modelo sugerido | Motivo |
|---|---|---|
| Pousada / Hospedagem | Tiny London, Tiny Monaco, Tiny Grécia | Visual mais hospitaleiro |
| Home office / Escritório | Office London, Office Venus | Proposição clara de uso |
| Experiência gourmet/bar | Tiny Adega, Tiny Gourmet | Nicho de experiências |
| Espaço saúde/wellness | Tiny Sauna | Proposição única |
| Moradia compacta | Tiny Natura, Tiny Atenas, Casa Nativa | Residencial real |
| Hospedagem premium | Casa Nativa (fotos reais), Casa Marrocos | Prova real + aspecto premium |

### Qualidade e uso técnico
- Todos os renders estão em alta resolução — adequados para impressão
- Formatos: PNG (fundo transparente em alguns casos) e JPG
- Para redes sociais: recortar para 1:1 (feed) ou 9:16 (stories/reels)
- Para apresentações: manter proporção original

## 03_Portfolio_Construcao

### Hierarquia de uso

**Tier 1 — Máximo impacto visual (usar primeiro)**
- `CASA_VIANA` — 170 fotos profissionais, qualidade editorial
- `CASA_BOSQUE_MARCIO` — 112 fotos, inclui externas + interiores premium

**Tier 2 — Projetos completos com boa documentação**
- `TERRAS_ALTAS` — 91 fotos, projeto amplo
- `VIVENDAS_DA_SERRA` — 70 fotos
- `LUMINOTECA_MALDANER` — 40 fotos, parceria com arquiteto

**Tier 3 — Projetos pontuais / MCMV**
- `LOTEAMENTO_CANELA`, `OBRA_CANELA_ALPES`, `OBRA_NOVA_PETROPOLIS` — bons para escala
- `CASA_SL_G17`, `CASA_SL_G23` — obras de médio porte
- `Casas_MCMV_Mucum`, `Casas_MCMV_Bom_Retiro`, `Casa_MCMV_Cruzeiro` — prova de escala B2B

### Seleção por objetivo

| Objetivo | Fotos recomendadas |
|---|---|
| Pitch para cliente final B2C | CASA_VIANA (hero) + CASA_BOSQUE (interiores) |
| Apresentação para prefeitura/governo | MCMV Muçum + MCMV Bom Retiro + escala |
| Material para arquitetos | LUMINOTECA_MALDANER + CASA_VIANA + CASA_BOSQUE |
| Pitch para licenciado | Mix de projetos por categoria (variedade conta) |
| Redes sociais — portfólio | CASA_VIANA para feed principal; obras em andamento para stories |
| Proposta comercial | 3-5 fotos: 1 externa frontal + 1 detalhe + 1 interior |

### Regras de uso
- ✅ Usar sempre crédito quando disponível (ex: *"Fotografia: Clara Beatriz — Entreponto Arq"* na CASA VIANA)
- ✅ Mostrar diversidade de projetos em materiais institucionais
- ❌ Nunca usar fotos de obra inacabada como imagem principal de venda
- ❌ Nunca editar fotos de clientes sem autorização (recortes são OK)

## Combinando assets em materiais

### Regra fundamental
- Logos Construção + Fotos Portfólio + Paleta preto/laranja → **material de Construção**
- Logos Tiny House + Renders Tiny House + Paleta marrom/bege → **material de Tiny House**
- ❌ Nunca misturar fotos de Tiny House em material de Construção e vice-versa

### Para apresentações completas do Grupo
- Abrir com identidade visual do Grupo Comsteel (preto/neutro)
- Apresentar Construção & Mobilidade com sua paleta
- Apresentar Tiny House com sua paleta
- Fechar com identidade do grupo novamente
- Manter separação visual clara entre as duas unidades
