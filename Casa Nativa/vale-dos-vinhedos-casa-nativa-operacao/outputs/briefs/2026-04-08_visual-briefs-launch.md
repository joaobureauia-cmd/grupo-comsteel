## Nota de Aplicação para o Designer

Todos os prompts acima foram desenvolvidos para gerar imagens de referencia via Midjourney ou DALL-E. O designer deve использовать os prompts como ponto de partida, adaptando para photographing os materiais reais da Casa Nativa cuando disponíveis.

**Paleta de segurança para tratamento de imagem:**
- Tons permitidos: #80421c, #945128, #e8e7e7, #f5ede0, #c4a882, #5c3d22, #faf8f4
- Tons proibidos: qualquer tom frio isolado (azul puro, verde vibrante, roxo)
- Filtro base: Warming filter ou tratamento manual para manter temperatura quente

**Câmera e lente recomendadas (se fotografia real):**
- Sony A7R IV ou Canon EOS R5
- Lentes: 35mm (arquitetura exterior), 50mm ou 85mm (retrato/interior), 90mm macro (detalhes)
- Filtro: UV combeschutzer ou polarizador para kontrol reflections nos brises bronze
