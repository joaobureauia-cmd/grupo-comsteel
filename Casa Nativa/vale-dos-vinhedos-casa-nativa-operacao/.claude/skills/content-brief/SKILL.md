# Skill: content-brief-generator
## Geração de Briefs Visuais para o Designer — Casa Nativa

### Quando usar esta skill
Use quando `content-brief-generator` precisar gerar um brief completo para o designer executar.
Aplicável a: posts de feed, reels, stories, criativos de anúncio, thumbnails.

---

### Protocolo de Execução

**Antes de gerar qualquer brief:**
1. Aplique a skill `brand-context` (leia o checklist de conformidade)
2. Confirme o tipo de conteúdo (feed / reels / stories / ad)
3. Confirme o objetivo (engajamento / alcance / conversão / branding)

---

### Template de Brief Visual

Use este template para cada entrega ao designer:

```
═══════════════════════════════════════════
BRIEF VISUAL — CASA NATIVA
═══════════════════════════════════════════

📅 DATA DE ENTREGA SUGERIDA: [data]
📱 CANAL: [Instagram Feed / Stories / Reels / Meta Ad]
🎯 OBJETIVO: [Engajamento / Alcance / Conversão / Branding]
📐 FORMATO(S): [Tamanhos em px — ex: 1080x1080, 1080x1920]

───────────────────────────────────────────
1. CONCEITO / IDEIA CENTRAL
───────────────────────────────────────────
[Descreva em 2-3 frases o conceito que guia este conteúdo.
O que queremos que o público sinta ao ver essa peça?]

───────────────────────────────────────────
2. TEXTO / COPY (já aprovado)
───────────────────────────────────────────
HEADLINE (se houver): "[texto]"
SUBTÍTULO (se houver): "[texto]"
CTA: "[ex: Reserve sua estadia]"
TAGLINE (se aplicável): "Um refúgio para desacelerar"

───────────────────────────────────────────
3. REFERÊNCIA DE IMAGEM / FOTO
───────────────────────────────────────────
Usar foto de: [descrever qual cena — ex: sauna ao entardecer, sala com lareira acesa,
deck com vista para o jardim. Indicar o link do Google Drive se aplicável]

Estilo fotográfico obrigatório:
- Luz quente e filtrada (dourada, ao entardecer/amanhecer)
- Nunca luz fria ou estourada
- Privilegiar texturas naturais (madeira, pedra, folhagem)
- Composição: natureza integrada ao espaço

───────────────────────────────────────────
4. PALETA DE CORES A USAR
───────────────────────────────────────────
[ ] Combinação 1 — Primária: fundo #f5ede0 + texto #80421c + destaque #945128
[ ] Combinação 2 — Impacto: fundo #80421c + texto #ffffff + destaque #e8e7e7
[ ] Combinação 3 — Cards: fundo #faf8f4 + texto #5c3d22 + CTA #945128
[ ] Combinação 4 — Ads: fundo #ffffff + texto #000000 + CTA #80421c
Combinação selecionada: [indicar qual]

───────────────────────────────────────────
5. TIPOGRAFIA
───────────────────────────────────────────
- Títulos/Headlines: Perandory (Google Fonts) — peso regular ou semibold
- Corpo/Subtítulos: Helvetica World ou Inter
- Tagline/Destaque poético: Playfair Display Italic
[Indicar qual hierarquia usar nesta peça específica]

───────────────────────────────────────────
6. ELEMENTOS VISUAIS / MOOD
───────────────────────────────────────────
Incluir (marcar os que se aplicam):
[ ] Ripados/brises como elemento decorativo
[ ] Vegetação/folhagem nativa integrada
[ ] Textura de madeira ou pedra
[ ] Geometria modular discreta (sugere os módulos)
[ ] Overlay de cor semi-transparente sobre foto
[ ] Linha/traço em terra escura (#80421c) como detalhe

Mood: [descrever o estado emocional que a peça deve transmitir]
Ex: "silêncio e aconchego ao final do dia", "leveza de manhã entre vinhedos"

───────────────────────────────────────────
7. HIERARQUIA DE MARCAS NESTA PEÇA
───────────────────────────────────────────
Logo Casa Nativa: [posição — ex: canto inferior direito, centralizado]
Logo Comsteel (se aplicável): [marca d'água discreta]
Logo Alinea (se aplicável): [onde e tamanho]

───────────────────────────────────────────
8. O QUE EVITAR
───────────────────────────────────────────
- Cores frias isoladas (azul, verde vibrante, roxo)
- Luz fria ou estourada nas fotos
- Fontes diferentes das oficiais
- Excesso de texto na arte
- Visual muito "hoteleiro" ou corporativo — manter orgânico e artesanal

───────────────────────────────────────────
9. REFERÊNCIAS VISUAIS EXTERNAS (se houver)
───────────────────────────────────────────
[Links ou descrições de referências do mercado para inspiração]

───────────────────────────────────────────
10. OBSERVAÇÕES ADICIONAIS
───────────────────────────────────────────
[Qualquer instrução específica para esta peça]

═══════════════════════════════════════════
✅ Conformidade de marca verificada — brand-guardian
═══════════════════════════════════════════
```

### Formatos de Arquivo por Canal
| Canal | Formato | Tamanho |
|-------|---------|---------|
| Instagram Feed (quadrado) | JPG/PNG | 1080 x 1080px |
| Instagram Feed (retrato) | JPG/PNG | 1080 x 1350px |
| Instagram Stories / Reels | JPG/PNG/MP4 | 1080 x 1920px |
| Meta Ad (feed) | JPG/PNG | 1080 x 1080px ou 1200 x 628px |
| Meta Ad (stories) | JPG/PNG | 1080 x 1920px |

---

### Salvamento
Salvar briefs em `outputs/briefs/` com nomenclatura:
`AAAA-MM-DD_brief-[tipo]-[tema].docx`
