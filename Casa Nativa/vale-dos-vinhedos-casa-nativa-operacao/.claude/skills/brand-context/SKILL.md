# Skill: casa-nativa-brand-context
## Injeção de Contexto de Marca — Casa Nativa

### Quando usar esta skill
Use esta skill **sempre** antes de criar qualquer conteúdo, copy, brief visual ou campanha.
É o ponto de partida obrigatório para todos os agentes de criação.

---

### Protocolo de Execução

**Passo 1 — Carregar contexto**
Leia o arquivo `context/context-casa-nativa.md` completamente antes de iniciar qualquer produção.

**Passo 2 — Extrair os elementos relevantes para a tarefa**
Identifique qual seção do contexto é prioritária para a tarefa:
- Criação de copy → Tom de Voz + Posicionamento + Palavras-chave
- Criação visual / brief → Identidade Visual + Paleta + Tipografia
- Conteúdo de canal específico → Canais de Distribuição + hierarquia de marcas
- Airbnb / Landing Page → Produto Real da Hospedagem + Diferenciais

**Passo 3 — Aplicar checklist de conformidade**
Antes de entregar qualquer output, valide:

#### ✅ CHECKLIST DE MARCA
- [ ] Tom sensorial, acolhedor e contemplativo? (não genérico, não clichê)
- [ ] Palavras proibidas ausentes? ("paraíso", "incrível", "melhor do mundo")
- [ ] Tiny Adega mencionada apenas se for contexto de brand story? (nunca como comodidade)
- [ ] Sauna privativa destacada como diferencial principal?
- [ ] Hierarquia de marcas correta? (Casa Nativa = principal)
- [ ] Paleta de cores respeitada? (sem tons frios isolados)
- [ ] Tipografia correta? (Perandory títulos / Helvetica World corpo / Playfair Display taglines)
- [ ] Tagline usada corretamente? ("Um refúgio para desacelerar")

**Passo 4 — Entregar com assinatura de conformidade**
Ao final de qualquer output, adicione:
```
✅ Conformidade de marca verificada — brand-guardian
```
