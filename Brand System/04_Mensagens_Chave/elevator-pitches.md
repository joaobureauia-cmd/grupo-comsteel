## Estrutura de Pitch Aprovada

Todo pitch deve seguir esta estrutura:

```
1. DADO DE AUTORIDADE     → número ou fato que para a atenção
2. DIAGNÓSTICO            → pergunta que mapeia a dor do interlocutor
3. CONSEQUÊNCIA           → o que o problema está custando (não o produto ainda)
4. SOLUÇÃO + PROCESSO     → como a Comsteel resolve, com prova
5. PRÓXIMO PASSO          → claro, específico, sem pressão
```
