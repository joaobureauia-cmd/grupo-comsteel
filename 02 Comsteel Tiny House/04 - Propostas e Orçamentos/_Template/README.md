# Template de Proposta — Comsteel Tiny House

Engine reutilizável para gerar propostas comerciais em PDF (vertical, A4, 10 páginas) com a identidade visual da Comsteel Tiny House.

---

## Como pedir uma nova proposta

Envie ao Claude as informações abaixo (apenas os blocos que mudam — o que não vier preenchido será mantido como o padrão Natura/Rodrigo).

A maneira mais rápida é copiar o **formulário** abaixo, preencher e mandar junto com as fotos.

## Como gerar a proposta (após receber as informações)

1. **Editar o `CONFIG`** no topo de `build_proposta.py` com os dados do novo cliente.
2. **Substituir as fotos** em `assets/` ou ajustar o `CONFIG["imagens"]` para apontar para os novos arquivos.
3. **Rodar:** `cd "_Template" && python3 build_proposta.py`
4. O PDF será gerado no nome definido em `CONFIG["output"]` (ex.: `Proposta_Tiny_House_<Modelo>_<Cliente>.pdf`).
5. **Mover** o PDF final para a pasta `04 - Propostas e Orçamentos/` (saindo do `_Template`).

> **Não editar o template em `_Template/`.** Faça uma cópia da pasta para o novo cliente, ou apenas peça a proposta ao Claude que ele cria a partir deste template.
