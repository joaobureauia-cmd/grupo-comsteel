# Hub de Arquitetos — Comsteel

## O que é este hub

Este diretório documenta o programa **Arquiteto Sócio-Construtivo** da Comsteel — o ecossistema de captação, formação e parceria estratégica com arquitetos e engenheiros.

O programa opera em **dois eixos simultâneos:**

| Eixo | Objetivo | Resultado |
|---|---|---|
| **Geração de negócio** | Arquitetos trazem projetos para dentro da Comsteel | Ampliação do pipeline comercial via rede de profissionais |
| **Desenvolvimento de cluster** | Comsteel seleciona e forma os "10 Arquitetos Modulares" | Cluster de referência em construção industrializada no RS/SC |

## Como usar este hub

- **Time comercial:** leia `captacao/pitch_para_arquitetos.md` e `captacao/cadencia_outbound.md` antes de abordar qualquer arquiteto
- **Gestor do programa:** leia `programa/programa_socio_construtivo.md` e `programa/selecao_10_modulares.md` para entender os critérios e o processo de onboarding
- **Apresentação ao arquiteto:** use `programa/sociedade_construtiva.md` como base do deck de apresentação
- **Treinamento:** `treinamento/steel_select_arquitetos.md` define o currículo e cronograma do Steel Select para esse público
