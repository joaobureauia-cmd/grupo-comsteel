# Seleção: Os 10 Arquitetos Modulares

## O que são os Arquitetos Modulares

Os **10 Arquitetos Modulares** são o núcleo do cluster da Sociedade Construtiva — os arquitetos selecionados para integrar o programa completo da Comsteel. Não é um título honorífico. É uma seleção funcional: esses são os profissionais com quem a Comsteel vai construir o ecossistema de construção industrializada no RS e SC.

> O limite é deliberado: 10 a 20 arquitetos. Não dá para dar visibilidade, indicações e formação de qualidade para um grupo infinito. Quem está dentro, está dentro de verdade.

---

## Critérios de Seleção

### Critérios eliminatórios (precisa ter)

| Critério | Detalhe |
|---|---|
| CAU ativo | Registro no Conselho de Arquitetura e Urbanismo |
| Atuação em residencial de médio a alto padrão | Perfil de cliente que consome Steel Frame |
| Localização prioritária | RS ou SC (expansão para outros estados com licenciados) |
| Escritório ativo | Mínimo 1-2 projetos/trimestre — não pode ser inativo |
| Abertura para aprender Steel Frame | Não precisa dominar — precisa ter disposição real |

### Critérios de priorização (diferenciam candidatos)

| Critério | Por que importa |
|---|---|
| Presença digital ativa | Instagram ou LinkedIn com portfólio — vai amplificar a visibilidade conjunta |
| Rede de clientes estabelecida | Mais projetos potenciais desde o início |
| Frustração com executores atuais | Maior abertura para o modelo Comsteel |
| Localização estratégica | Cobertura de praças prioritárias (POA, Caxias, Gramado, SC) |
| Experiência com projetos de múltiplas unidades | Condomínios, loteamentos, pousadas — ticket maior |

## O que o Arquiteto Modular recebe

| Benefício | Detalhe |
|---|---|
| Steel Select completo | Treinamento em construção industrializada (Steel Frame, Tiny House) — acesso prioritário |
| Indicações de clientes da Comsteel | A Comsteel indica o arquiteto para leads que precisam de projeto |
| Presença nos materiais da marca | Nome, portfólio e link nos materiais Comsteel (site, catálogo, Instagram) |
| Comissão sobre obras executadas | 2% (indicação simples) a 4% (parceria construtiva) sobre o valor do contrato |
| Participação nos cases | O arquiteto assina as obras que acompanhou — visibilidade com autoria |
| Suporte técnico direto | Acesso direto a Felipe (técnico) e equipe para dúvidas de projeto em Steel Frame |
| Vantagem competitiva temporal | Entra no segmento antes de ele estar saturado de especialistas |

## Processo de Seleção

### Fase 1 — Identificação (outbound)
- Prospecção ativa via Instagram, LinkedIn e eventos
- Filtro inicial: CAU ativo, residencial médio-alto, RS/SC
- Meta: 30-50 candidatos identificados no primeiro trimestre

### Fase 2 — Primeiro contato e diagnóstico
- Abordagem via script de diagnóstico (ver `captacao/pitch_para_arquitetos.md`)
- Objetivo: entender a realidade atual do arquiteto ANTES de apresentar o programa
- Pergunta central: "Qual é o maior problema que você tem hoje entre projeto aprovado e execução entregue?"

### Fase 3 — Apresentação do ecossistema
- Envio do material da Sociedade Construtiva
- Conversa de 20-30 minutos (presencial ou video) com Ramão ou João
- Visita a obra em andamento ou showroom (alta conversão)

### Fase 4 — Proposta de entrada
- Proposta formal de parceria com escopo claro
- Assinatura do protocolo de parceria (documento de relacionamento com cliente)
- Agendamento do Steel Select

### Fase 5 — Onboarding
- Inclusão nos materiais da Comsteel
- Acesso ao Steel Select
- Primeiro projeto piloto com acompanhamento próximo
- Check-in em 30 dias pós-entrada

## O que NÃO é o Arquiteto Modular

- Não é um indicador passivo que manda contatos ocasionais
- Não é alguém que "vai ver se tem projeto"
- Não é um cliente — é um parceiro com papel ativo no ecossistema
- Não é exclusivo (pode trabalhar com outras empresas), mas é comprometido com o processo Comsteel quando o projeto envolve Steel Frame
